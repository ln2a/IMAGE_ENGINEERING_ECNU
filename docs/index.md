## 课程内容概览

本课程涵盖图像工程的核心内容，共分为五个部分：

1. **第一部分：图像形成与数字图像基础** — 成像模型、正问题与逆问题、图像先验
2. **第二部分：图像增强与滤波** — 灰度变换、直方图均衡化、均值/高斯/中值滤波
3. **第三部分：锐化与边缘检测** — 梯度算子、Sobel 算子、拉普拉斯算子
4. **第四部分：图像分割** — 阈值分割、区域生长、语义分割
5. **第五部分：图像特征与目标识别** — 颜色/纹理/形状特征、CNN、目标检测

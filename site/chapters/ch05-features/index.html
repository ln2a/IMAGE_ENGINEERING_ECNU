    第五部分 图像特征与目标识别¶

    第五部分：图像特征与目标识别¶

    核心问题

    经过增强、边缘检测和分割之后，我们希望进一步回答：

    图像中有什么？它属于哪一类？

    从区域到理解

    图像 \(\longrightarrow\) 特征提取 \(\longrightarrow\) 分类识别

    图像特征

    • 描述颜色
    • 描述纹理
    • 描述形状
    • 描述局部结构

    目标识别

    • 判断图像类别
    • 判断区域类别
    • 识别物体名称
    • 支持后续决策

    一句话

    特征是图像内容的 “描述方式”，识别是根据这些描述判断 “它是什么”。

    什么是图像特征？¶

    直观理解

    图像特征就是从图像中提取出来的、有助于区分不同目标的信息。

    人是怎样识别物体的？

    • 颜色：香蕉是黄色的
    • 形状：车轮大多是圆形的
    • 纹理：树皮有粗糙纹理
    • 结构：人脸有眼睛、鼻子和嘴

    计算机需要什么？

    • 把图像转成数值描述
    • 找出稳定、有区分性的模式
    • 用这些模式进行分类或匹配

    关键理解

    图像特征不是图像本身，而是对图像中有用信息的概括。

    为什么不能直接用像素识别？¶

    一个朴素想法

    既然图像本来就是像素矩阵，能不能直接比较像素？

    \[ I = \left[ \begin{array}{c c c} I (1, 1) & I (1, 2) & \dots \\ I (2, 1) & I (2, 2) & \dots \\ \vdots & \vdots & \ddots \end{array} \right] \]

    问题在于：像素太敏感

    • 物体稍微平移，像素位置就变了；
    • 光照稍微变化，灰度值就变了；
    • 拍摄角度变化，形状和纹理也会变化；
    • 背景复杂时，很多像素与目标无关。

    一句话

    直接比较像素往往不稳定，因此需要提取更加稳健的图像特征。

    颜色特征：最直观的图像描述¶

    基本思想

    颜色特征描述图像中颜色的组成和分布。

    常见方法：颜色直方图

    统计不同颜色或灰度在图像中出现的频率。

    \[ h (k) = \# \{(x, y) \mid I (x, y) = k \} \]

    优点

    • 简单直观；
    • 对物体位置变化不太敏感；
    • 适合描述颜色差异明显的图像。

    缺点

    • 不能很好描述空间结构；
    • 不同物体可能有相似颜色；
    • 容易受光照影响。

    例子：只看颜色，绿色苹果和绿色葡萄可能很难区分。

    纹理特征：描述局部重复结构¶

    什么是纹理？

    纹理是图像中局部灰度或颜色反复变化形成的结构模式。

    常见纹理

    • 木纹
    • 布料纹理
    • 砖墙纹理
    • 草地纹理
    • 医学图像中的组织纹理

    纹理能描述什么？

    • 粗糙还是平滑
    • 是否有周期性
    • 方向是否明显
    • 局部结构是否规则

    理解

    纹理特征关注的不是单个像素，而是一小片区域内的灰度变化规律。

    形状特征：描述目标轮廓¶

    基本思想

    形状特征用于描述目标区域的几何外观。

    常见形状信息

    • 面积
    • 周长
    • 长宽比
    • 圆形度
    • 边界曲率

    适用场景

    • 工业零件检测
    • 医学病灶分析
    • 字符识别
    • 目标轮廓匹配

    例子

    圆形、细长形、不规则形状，往往对应不同类型的目标。

    注意

    形状特征通常依赖较好的分割结果；如果分割不准，形状描述也会受到影响。

    局部特征：抓住图像中的关键位置¶

    基本思想

    有些位置比普通像素更有辨识度，例如角点、斑点、纹理突变处。

    局部特征关注什么？

    • 哪些位置比较稳定？ 这些位置周围长什么样？
    • 不同图像中能否找到对应关系？

    典型应用

    • 图像匹配；
    • 全景拼接；
    • 目标跟踪；
    • 三维重建；
    • 视觉定位。

    一句话

    局部特征就像图像中的 “指纹点”，可以帮助计算机在不同图像之间建立对应关系。

    从人工特征到深度特征¶

    传统方法

    人工设计特征，然后交给分类器判断。

    图像 \(\longrightarrow\) 人工特征 \(\longrightarrow\) 分类器 \(\longrightarrow\) 类别

    深度学习方法

    神经网络自动从数据中学习特征。

    图像 \(\longrightarrow\) 神经网络 \(\longrightarrow\) 类别

    人工特征

    • 可解释性较强
    • 设计依赖经验
    • 复杂场景表达能力有限

    深度特征

    • 表达能力强
    • 可自动学习
    • 依赖大量数据和算力

    目标识别：从特征到类别¶

    基本思想

    目标识别的任务，是根据图像或图像区域的特征，判断它属于哪一类。

    图像区域 \(\longrightarrow\) 特征表示 \(\longrightarrow\) 分类结果

    输入

    • 一整幅图像；
    • 一个分割出来的区域；
    • 一个检测框中的目标；
    • 一个局部图像块。

    输出

    • 是否有人脸；
    • 是猫还是狗；
    • 是正常还是异常；
    • 是道路、车辆还是行人。

    一句话

    目标识别就是让计算机根据图像内容回答 “这是什么”。

    分类器：根据特征做判断¶

    传统识别流程

    通常先提取特征，再使用分类器进行判断。

    图像 \(\longrightarrow\) 特征向量 \(\longrightarrow\) 分类器 \(\longrightarrow\) 类别

    特征向量

    把颜色、纹理、形状等信息表示成一组数字：

    \[ \mathbf {x} = (x _ {1}, x _ {2}, \dots , x _ {d}) \]

    分类器的作用

    • 学习不同类别的特征规律；
    • 根据输入特征判断类别；
    • 给出分类结果或置信度。

    理解

    分类器可以看作一个决策规则：输入一组特征，输出一个类别。

    一个简单分类例子：猫和狗¶

    问题

    给定一张动物图片，希望判断它是猫还是狗。

    传统方法可能会怎么做？

    • 提取颜色、纹理特征、耳朵、脸型、轮廓等形状特征；
    • 把这些特征输入分类器。

    动物图像 \(\longrightarrow\) 特征 \(\longrightarrow\) 猫 / 狗

    困难在哪里？

    • 同一类动物外观差异很大；
    • 姿态、角度、光照会变化；
    • 背景可能很复杂；
    • 有时局部遮挡会影响判断。

    目标识别为什么困难？¶

    同一个目标可能看起来很不一样

    • 角度变化：正面、侧面、俯视；
    • 尺度变化：远处小，近处大；
    • 光照变化：白天、夜晚、阴影；
    • 姿态变化：人站立、奔跑、弯腰；
    • 遮挡变化：目标可能只露出一部分。

    不同目标也可能看起来很相似

    • 猫和小狗；
    • 普通痣和皮肤病灶；
    • 正常零件和轻微缺陷零件；
    • 遥感图像中的道路和屋顶边缘。

    关键理解

    目标识别不是简单地比较像素，而是要在复杂变化中抓住稳定的本质特征。

    卷积神经网络：自动学习图像特征¶

    基本思想

    卷积神经网络 CNN 可以从大量图像中自动学习多层次特征。

    图像 \(\longrightarrow\) 低层特征 \(\longrightarrow\) 中层特征 \(\longrightarrow\) 高层语义 \(\longrightarrow\) 类别

    低层特征

    • 边缘
    • 角点
    • 颜色变化

    中层特征

    • 局部纹理
    • 轮廓
    • 部件结构

    高层特征

    • 人脸
    • 车辆
    • 器官
    • 场景语义

    一句话

    深度网络不是手工指定特征，而是从数据中逐层学习特征。

    卷积操作：局部模式检测¶

    基本思想

    卷积核在图像上滑动，检测局部区域中是否存在某种模式。

    \[ g (i, j) = \sum_ {m, n} w (m, n) f (i - m, j - n) \]

    和前面滤波的联系

    • 平滑滤波：卷积核用于平均或加权平均；
    • 边缘检测：卷积核用于检测灰度变化；
    • CNN：卷积核通过训练自动学习。

    重要联系

    卷积神经网络中的卷积，与前面讲过的滤波、锐化、边缘检测在形式上有相通之处。

    深度网络为什么有效？¶

    核心原因

    深度网络能够学习从简单到复杂的多层次表示。

    层次化表示

    像素 → 边缘 → 纹理 → 部件 → 目标

    优势

    • 自动学习特征，不完全依赖人工设计；
    • 能表达复杂外观变化；
    • 可以利用大规模数据提升性能；
    • 在分类、检测、分割等任务中表现突出。

    但也要注意

    深度学习效果好，并不意味着它不需要数据质量、模型设计和结果解释。

    图像分类、目标检测与分割的区别¶

    三类常见视觉任务

    图像分类

    图像 \(\rightarrow\) 类别

    • 判断整幅图是什么； 例如：猫、狗、汽车。

    目标检测

    图像 \(\rightarrow\) 框 \(+\) 类别

    • 找到目标在哪里；
    • 同时判断目标类别。

    图像分割

    图像 \(\rightarrow\) 像素类别

    • 判断每个像素属于什么；
    • 得到更精细的区域。

    一句话

    分类回答 “是什么”，检测回答 “在哪里、是什么”，分割回答 “每个像素是什么”。

    目标检测：不仅识别，还要定位¶

    基本任务

    目标检测要在图像中找出目标的位置，并判断目标类别。

    图像 \(\longrightarrow\) {(位置框，类别，置信度)}

    输出通常包括

    • 边界框：目标大致在哪里；
    • 类别标签：目标是什么；
    • 置信度：模型有多确定。

    应用场景

    • 自动驾驶中检测车辆、行人和交通标志；
    • 工业检测中定位缺陷区域；
    • 医学图像中辅助定位病灶；
    • 安防监控中检测异常目标。

    识别结果如何评价？¶

    分类任务常用指标

    • 准确率 Accuracy: 整体分对了多少；- 精确率 Precision：判为某类的结果中有多少是真的；
    • 召回率 Recall：真实属于某类的样本中找回了多少；
    • F1 值：综合考虑精确率和召回率。
    \[ \text { Accuracy } = \frac {\text { 预测正确的样本数 }}{\text { 总样本数 }} \]

    为什么不能只看准确率？

    如果异常样本很少，一个模型即使全部预测为正常，也可能有很高准确率，但它没有真正发现异常。

    工程理解

    评价指标要结合任务目标选择，不能只看一个数字。

    医学图像识别中的例子¶

    任务

    根据医学图像判断是否存在病灶，或者判断病灶类型。

    可能的输入

    • X 光图像；
    • CT 图像；- MRI 图像；
    • 病理切片图像。

    可能的输出

    • 正常 / 异常；
    • 良性 / 恶性；
    • 病灶位置；
    • 风险等级。

    注意

    医学图像识别不能只追求高准确率，还要重视可靠性、可解释性和临床验证。

    遥感图像识别中的例子¶

    任务

    从遥感图像中识别地物类型或目标对象。

    常见目标

    • 建筑物；
    • 道路；
    • 河流；
    • 农田；
    • 森林；
    • 船只、车辆等小目标。

    主要困难

    • 目标尺度变化大；
    • 拍摄角度和分辨率不同；
    • 背景复杂；
    • 同类地物差异明显；
    • 不同类地物外观相似。

    理解

    遥感图像识别不仅要看局部纹理，还要结合空间分布和场景上下文。

    第五部分阶段小结¶

    1 图像特征是对图像有用信息的数值化描述。 ② 颜色、纹理、形状和局部结构都是常见特征。 3 直接比较像素往往不稳定，需要更稳健的特征表示。 4 传统方法通常先人工设计特征，再使用分类器识别。 ⑤ 深度学习方法可以从数据中自动学习多层次特征。 6 图像分类、目标检测和图像分割回答的问题不同。 7 评价识别结果时，要根据任务选择合适指标。

    本部分核心

    从 “图像是什么样子” 到 “图像里有什么”，关键在于建立有效的图像表示。
